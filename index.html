Google Calendar
Let Ro see your calendar so it knows what your day looks like. Requires a Google Cloud OAuth Client ID.
To get a Client ID: console.cloud.google.com → New Project → Enable "Google Calendar API" → Credentials → OAuth 2.0 Client ID → Web application → add your hosted URL as Authorized Origin.
Install as Phone App
To install Ro on your phone:
1. Upload these files to GitHub Pages (free, takes 5 min)
2. Open the URL in Chrome (Android) or Safari (iPhone)
3. Android: tap ⋮ → "Add to Home screen"  |  iPhone: tap Share → "Add to Home Screen"

Your data stays private — it never leaves your device.
